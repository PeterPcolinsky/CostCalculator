# Cost Calculator

A simple Java console application for managing incomes and expenses.

## Features:
- Add income
- Add expense
- Show current balance (incomes - expenses)
- Exit the program

## Usage:
1. Run the program.
2. Choose an option from the menu:
   - 1 - Add income
   - 2 - Add expense
   - 3 - Show current balance
   - 0 - Exit the program

## Technologies:
- Java
- IntelliJ IDEA
- Git & GitHub
